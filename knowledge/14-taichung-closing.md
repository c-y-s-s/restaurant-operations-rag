# 台中店打烊補充 SOP

本文件為台中店專用打烊補充流程，須搭配全店打烊共同流程執行。台中店打烊時須特別確認最後點餐、兒童遊戲區、垃圾集中、瓦斯桶主閥與壓力表回報。

---

# 最後點餐時間

台中店最後點餐時間如下：

1. 週一至週五：21:00
2. 週六與週日：21:30

最後點餐時間前，前場人員應提醒候位與晚到客人，避免客人入座後才得知無法點餐。

最後點餐後，廚房與前場須確認未出餐項目、客製化需求與結帳狀態。若仍有客人用餐，不得催促離場，但可依標準話術提醒營業時間。

---

# 兒童遊戲區關閉與消毒

兒童遊戲區於每日 20:30 關閉並執行消毒。關閉後不得再開放客人進入。

消毒流程如下：

1. 確認遊戲區內無兒童或客人物品
2. 移除明顯垃圾與異物
3. 擦拭軟墊、護欄與常接觸表面
4. 檢查是否有破損、鬆動或髒污
5. 填寫清潔與消毒紀錄
6. 放置關閉或清潔完成標示

若發現設備破損或安全疑慮，須通知值班主管，並於隔日開店前再次確認是否可開放。

---

# 垃圾集中與保存

台中店垃圾須集中於停車場側門內的冷藏垃圾間，隔日由清運人員處理。不得提前放到戶外，以免產生異味、招引蟲害或違反場地規範。

垃圾集中前須確認：

1. 垃圾袋已確實綁緊
2. 廚餘與一般垃圾已依規定分類
3. 垃圾間門可正常關閉
4. 垃圾間溫度與環境無明顯異常
5. 搬運路線無滴漏或殘留物

若冷藏垃圾間異常或無法使用，須立即通知值班主管，不得自行改放戶外。

---

# 瓦斯桶主閥與壓力表回報

離店前，負責人員須關閉瓦斯桶主閥，並拍攝壓力表照片上傳至值班群組。

確認流程如下：

1. 關閉瓦斯桶主閥
2. 確認周邊無瓦斯味
3. 拍攝壓力表照片
4. 確認照片清楚可辨識
5. 上傳至值班群組
6. 回報值班主管完成狀態

若發現瓦斯味、壓力異常或閥門無法正常關閉，應立即遠離現場，避免開關電器，並通知值班主管處理。

---

# 離店前確認

值班主管離店前須確認以下事項：

1. 兒童遊戲區已關閉並完成消毒
2. 垃圾已集中於冷藏垃圾間
3. 瓦斯桶主閥已關閉
4. 壓力表照片已上傳值班群組
5. 側門、正門與相關入口已上鎖
6. 店內無人員滯留
7. 保全已成功設定

若任一項目未完成，不得直接離店，須完成補救或通知主管安排後續處理。
