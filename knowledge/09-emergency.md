# 緊急事故應變通則

本指南適用於全店發生火災、停電、地震或其他突發狀況時的初步應變。所有員工應以人員安全為優先，其次才是設備、食材與營運恢復。

發生緊急事故時，員工不得自行冒險處理超出能力範圍的狀況。若現場有立即危險，應先停止作業、遠離危險區域，並依值班主管指示疏散。

---

# 火災處理

若發生小型油鍋火災，應先關閉熱源，並使用 K 類滅火器或滅火毯處理。不得向油鍋火災潑水，以免造成油火噴濺與火勢擴大。

火災初期處理流程如下：

1. 立即關閉熱源
2. 大聲提醒周邊人員遠離
3. 使用適合的滅火設備處理
4. 通知值班主管
5. 確認火勢是否完全受控

若火勢無法立即控制、煙霧快速增加或有延燒風險，應立即按下警報、疏散客人與員工，並撥打 119。疏散時應引導人員前往店外指定集合點，不得返回店內拿取私人物品。

火災處理結束後，未經主管與相關單位確認，不得重新啟用受影響設備或恢復營業。

# 停電處理

停電發生時，員工應立即記錄停電開始時間，並通知值班主管。停電期間應保持冷藏與冷凍設備門關閉，避免溫度快速上升。

停電期間須停止製作需要加熱、冷藏保存或無法確保食品安全的餐點。已在製作中的餐點，須由主管依現場狀況判斷是否暫停、報廢或延後出餐。

停電處理流程如下：

1. 記錄停電時間
2. 停止高風險餐點製作
3. 保持冷藏與冷凍門關閉
4. 確認客人與員工動線安全
5. 通知主管評估營運狀況
6. 電力恢復後檢查設備與溫度紀錄

若停電超過兩小時，須由主管依冷藏、冷凍與食材溫度紀錄判斷是否報廢。員工不得自行恢復販售或重新標示受影響食材。

# 地震處理

地震搖晃期間，員工應遠離玻璃、吊掛物、高架層架與可能倒塌的設備，並就地掩護。廚房人員應在安全可行的情況下停止使用明火，但不得冒險穿越危險區域。

地震發生時應遵守以下原則：

1. 保護頭部與身體
2. 遠離玻璃與吊掛物
3. 不奔跑、不推擠
4. 不搭乘電梯
5. 避免靠近熱油、明火與重型設備

搖晃停止後，值班主管須確認人員傷勢、瓦斯狀態、設備狀態與逃生動線。若發現瓦斯味、漏水、火花、建築結構異常或設備損壞，應立即疏散並通知相關單位。

---

# 傷患與客人安撫

若有員工或客人受傷，應立即通知值班主管並視情況撥打 119。員工不得任意移動疑似骨折、昏迷或嚴重受傷者，除非現場有立即危險。

前場人員應以清楚、冷靜的語氣引導客人，不得散布未確認資訊。若需要疏散，應指引客人由最近安全出口離開，並協助年長者、兒童、行動不便者優先移動。

---

# 事故紀錄與復原

事故結束後，值班主管須完成事故紀錄，內容包含：

1. 發生日期與時間
2. 事故類型與發生位置
3. 現場處理方式
4. 是否有人員受傷
5. 是否有設備或食材受影響
6. 是否通報 119、維修單位或總公司
7. 後續改善事項

未完成安全確認前，不得恢復設備使用或重新開始營業。若涉及食品安全、設備故障或人員受傷，須依公司通報流程回報。
