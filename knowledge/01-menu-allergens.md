# 過敏原處理原則

員工不得向客人保證任何餐點為「完全無過敏原」或「絕對安全」。由於廚房設備、備料區、煎台與出餐流程可能與其他餐點共用，所有餐點皆存在交叉接觸風險。

當客人提出過敏申告時，員工應先確認以下資訊：

1. 過敏原名稱
2. 過敏嚴重程度
3. 是否曾有嚴重過敏反應
4. 是否可接受共用設備或微量交叉接觸風險

確認後，須在點餐系統中加註紅色過敏標籤，並口頭通知值班主管與出餐口。若客人曾有嚴重過敏反應，值班主管必須再次確認餐點食材標示、製作流程與交叉接觸風險後，才能繼續處理訂單。

# 花生與堅果

以下餐點含有花生或堅果類成分：

- 花生醬雞肉沙拉：含花生
- 青醬義大利麵：含腰果
- 提拉米蘇：表面裝飾可能使用杏仁片

香煎鮭魚與番茄蔬菜湯的標準配方不含花生與堅果，但由於廚房共用備料台與製作環境，仍無法排除微量交叉接觸的可能。若客人對花生或堅果有過敏疑慮，員工仍須依照過敏原處理原則進行確認與通報。

# 麩質與乳製品

以下餐點含有麩質或乳製品：

- 奶油蘑菇義大利麵：含小麥與乳製品
- 牛肉漢堡：麵包含小麥與奶油
- 提拉米蘇：含乳製品
- 香煎鮭魚：檸檬奶油醬含乳製品

番茄蔬菜湯的標準配方不含麩質與乳製品。香煎鮭魚可依需求將檸檬奶油醬改為鹽與橄欖油調味，但煎台與其他餐點共用，仍可能存在交叉接觸風險。

# 蛋與海鮮

以下餐點含有蛋或海鮮相關成分：

- 提拉米蘇：含蛋與乳製品
- 凱薩醬：含蛋與鯷魚
- 香煎鮭魚：含海鮮
- 海鮮燉飯：含海鮮
- 兒童番茄筆管麵：不含蛋，但麵體含小麥

若客人對蛋、魚類、甲殼類或其他海鮮成分有過敏疑慮，員工須依照過敏原處理原則確認過敏原種類與嚴重程度，並提醒客人共用設備可能造成交叉接觸風險。

# 對客溝通建議

員工可使用以下說法向客人說明：

「我們可以協助確認餐點的標準配方與主要過敏原成分，也會在系統中加註過敏提醒並通知主管與出餐口。不過，由於廚房設備與備料環境可能共用，我們無法保證任何餐點完全沒有交叉接觸風險。」

若客人表示曾發生嚴重過敏反應，應請值班主管介入確認，並避免由一般員工單獨承諾餐點安全性。
