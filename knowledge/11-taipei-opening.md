# 台北店開店補充 SOP

本文件為台北店專用開店補充流程，須搭配「全店開店共同流程」執行。若本文件與全店共同流程皆有規範，應同時遵守；若現場狀況有安全疑慮，應優先依值班主管指示處理。

---

# 時間與入口

台北店早班人員應於 09:30 從南京東路員工入口進入。進入後先依全店開店共同流程完成保全解除、安全巡檢與必要照明開啟。

台北店開店時程如下：

1. 09:30 早班人員進入店內
2. 10:00 前完成主要設備啟動
3. 10:30 後員工入口保持上鎖
4. 11:00 正式開始營業

10:30 後，員工入口應保持上鎖狀態。員工進出須使用個人感應卡，不得共用感應卡或讓未授權人員尾隨進入。

---

# 地下室抽水泵確認

開店時須額外開啟地下室抽水泵監控面板，確認面板綠燈恆亮。

檢查時應確認以下事項：

1. 監控面板電源正常
2. 綠燈恆亮，無閃爍或異常警示
3. 周邊無積水、異味或異常聲響
4. 面板位置與通道無障礙物

若綠燈未亮、出現警示、周邊積水或設備異常聲響，應立即通知值班主管。未確認狀況前，不得自行重啟或拆卸設備。

---

# 戶外座位擺設

台北店戶外座位僅在氣象預報降雨機率低於 60%，且現場風勢安全時擺設。

擺設戶外座位前，須確認以下事項：

1. 降雨機率低於 60%
2. 現場風勢不影響桌椅與陽傘安全
3. 地面無積水、油漬或障礙物
4. 桌椅乾淨且穩固
5. 動線不阻礙行人與出入口

若天候不穩、風勢過大或地面濕滑，應暫停擺設戶外座位，並由值班主管決定是否開放戶外區。

---

# 開店確認與回報

台北店早班人員完成專用項目後，須回報值班主管以下事項：

1. 員工入口已依規定管理
2. 主要設備已於 10:00 前啟動
3. 地下室抽水泵面板狀態正常
4. 戶外座位是否開放
5. 是否有設備、天候或安全異常

若任一項目未完成或出現異常，須於營業前會議中提醒全體當班人員。
