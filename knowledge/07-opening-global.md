# 全店開店共同流程

本流程適用於所有分店早班開店作業。早班人員須依照標準順序完成安全檢查、設備啟動、食材確認、環境準備與營業前會議，確保門店可安全、準時營業。

若開店過程中發現安全疑慮、設備異常、食材保存異常或人力不足，須立即通知值班主管，不得自行忽略或延後處理。

---

# 到店與安全檢查

早班人員到店後，應先解除保全系統，開啟必要照明，並確認店內外環境是否安全。

須優先檢查以下項目：

1. 門窗是否完整、無破損或遭破壞痕跡
2. 店內是否有異常氣味，尤其是瓦斯味或燒焦味
3. 地面是否有積水、油漬或其他滑倒風險
4. 電器、插座與設備外觀是否異常
5. 冷藏、冷凍與主要設備是否正常運作
6. 出入口、逃生通道與工作動線是否暢通

若發現瓦斯味，不得開啟或關閉任何電器、照明、排風設備或插座開關。應立即關閉瓦斯總閥，離開店內至安全位置，並通知值班主管或相關維修單位處理。

在安全狀況未確認前，不得讓員工進入廚房作業，也不得開始營業準備。

---

# 廚房設備啟動

確認店內安全後，依序啟動營業所需設備。設備啟動時須注意是否有異常聲響、異味、漏水、漏油、無法加熱或面板警示。

開店時須確認並啟動以下設備：

1. 排煙設備
2. 製冰機
3. 洗碗機
4. 冷藏與冷凍設備
5. 必要烹調設備
6. POS 系統與出單設備
7. 熱水、照明與其他營業必要設備

若設備無法正常啟動，應立即通知值班主管，並於設備異常紀錄中登記。未經主管確認，不得使用有異常疑慮的設備。

---

# 冷藏與冷凍溫度紀錄

開店時須記錄所有冷藏與冷凍設備溫度。

冷藏設備應維持在 0°C 至 7°C 之間，建議目標設定為 4°C。冷凍設備應維持在 -18°C 以下。

若溫度超出標準範圍，須立即重新測量一次。若複測後仍異常，應通知值班主管，並暫停使用受影響食材，直到主管完成確認。

受影響食材不得直接投入備料或販售。若無法確認食材是否仍符合安全標準，須依報廢流程處理。

---

# 食材標籤與效期檢查

開店後須檢查昨日留存食材、已開封食材與預先備製食材的標籤。

檢查項目包含：

1. 品名是否清楚
2. 開封或製備時間是否完整
3. 保存期限是否仍有效
4. 容器是否密封或加蓋
5. 食材是否有異味、變色、黏滑或腐敗跡象

逾期食材須立即隔離，不得繼續使用。任何逾期品、標示不清品或疑似變質食材，不得重新貼標、修改日期或混入新食材中。

若員工不確定食材是否可使用，須交由值班主管確認，不得自行判斷出餐。

---

# 前場與用餐區準備

前場人員須於營業前完成用餐區、櫃台與外帶區準備，確保客人進店時環境整潔、動線清楚。

營業前須確認以下項目：

1. 桌椅排列整齊
2. 桌面、地面與櫃台清潔完成
3. 菜單、餐具、紙巾與調味品補齊
4. 收銀機、刷卡機與發票設備正常
5. 外帶包材、號碼牌與取餐區準備完成
6. 門口、招牌與營業告示狀態正確

若有缺品、設備異常或環境清潔未完成，須於開店前回報值班主管安排補救。

---

# 廚房備料與出餐準備

廚房人員須依當日預估營業量完成備料，並確認各工作站備品充足。

營業前須完成以下準備：

1. 檢查各工作站食材是否齊全
2. 補足醬料、配菜、湯品與常用食材
3. 確認刀具、砧板、夾具與容器已清潔消毒
4. 確認生食、熟食與即食食品分區擺放
5. 確認垃圾桶、回收桶與廚餘桶位置正確
6. 確認洗手台、清潔用品與消毒用品可正常使用

備料過程中須遵守生熟食分離與食品安全規範，不得將未確認效期或保存狀態的食材投入製作。

---

# 營業前會議

營業前 15 分鐘，由值班主管召開簡短營業前會議。所有當班員工應停止非必要工作並參與會議。

會議內容應包含：

1. 當日訂位與預估客流
2. 缺貨品項與替代方案
3. 特殊活動、促銷或大型訂單
4. 當日過敏客註記與特殊餐點需求
5. 人力配置與各工作站負責人
6. 設備異常、食材異常或安全注意事項
7. 當日服務重點與出餐提醒

若有過敏客註記，主管須明確提醒前場、廚房與出餐口，並確認點餐系統中的備註方式一致。

---

# 開店前最終確認

正式營業前，值班主管須完成最後巡檢，確認門店已具備營業條件。

最終確認項目包含：

1. 店內安全狀況正常
2. 設備已啟動且無異常
3. 冷藏與冷凍溫度已完成紀錄
4. 食材標籤與效期已檢查
5. 前場、廚房與出餐區準備完成
6. 當日缺貨與特殊注意事項已通知全員
7. POS、出單、付款與外帶流程可正常運作

所有項目確認完成後，方可開放營業。若仍有重大安全、設備或食品安全疑慮，須由值班主管判斷是否延後開店或調整營業內容。
